Provides the main classes that allowing access to the basic functions of PowerGrid.

This package contains two entities: the Bot and the PowerGrid class.

The Bot provides PowerGrid's basic functionality in easy-to-use methods.

The PowerGrid class provides access to more advanced parts of PowerGrid, including the {@link powergrid.control.Mapper Mapper}, {@link powergrid.control.TaskManager TaskManager} and {@link powergrid.model.DestinationMap DestinationMap}.
